Questions

  1. What does the server in the client sever architecture do?
    The purpose of a server in the client server architecture is to host one or more server programs which share their resources with clients
  2. What does the client do?
    Unlike the server a client does not share any of it's resources, but requests a server's content or service function.
  3. What is an IP address?
    An internet Protocil adress is a numerical label assigned to each device connected to a computer network.
  4. What is DNS?
    The Domain Name System (or DNS for short) is a central part of the internet, providing a way to match website names to numbers. Everything connected to the internet has an IP adress made up of numbers
  5. What is the difference between the Internet and WWW?
    The difference between the internet and WWW is that internet is short for intern network. So every collection of computers sharing information over a local network is using the internet. WWW on the other hand is a global internet, where every computer is connected.
  6. Explain what HTML is
    Hyper Text Markup Language (HTML for short) is the basic building block of the web. It is a simple way of creating and linking togheter webpages.
